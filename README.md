# 💼 Job Portal Web Application (Ruby on Rails)

A simple Job Portal web application built using Ruby on Rails that allows users to create job listings and apply for jobs with resume upload functionality.

---

## 🚀 Features

### Job Management
- Create new job postings
- View all job listings
- Edit job details
- Delete job postings

### Application System
- Apply for jobs
- Submit name, email, and resume
- View all applications for each job

### Additional Features
- Resume upload using Active Storage
- Database integration using SQLite
- Clean MVC architecture

---

## 🛠️ Technologies Used

- Ruby
- Ruby on Rails
- SQLite3
- HTML / ERB
- Bootstrap (basic styling)
- Active Storage (for file upload)

--



## ⚙️ How to Run This Project

```bash
git clone https://github.com/your-username/job-portal.git
cd job-portal
bundle install
rails db:migrate
rails server
```

Then open:
```
http://localhost:3000
```
