# Job Portal Web App

A simple Job Portal Web Application built using Ruby on Rails.

## Features
- Add Job Listings
- View Jobs
- Edit Jobs
- Delete Jobs
- Database Integration using SQLite
- Responsive UI using Bootstrap

## Technologies Used
- Ruby
- Ruby on Rails
- SQLite
- Bootstrap


## Future Improvements
- User Authentication
- Resume Upload
- Job Search
- Admin Dashboard
- Email Notifications
